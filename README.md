# @sencinion/ms

Türk kullanıcıları için yapılmış türkçe/ingilizce desteki ms modülü

## Kurulum

```bash
npm install @sencinion/ms
```

## Kullanım

```javascript
const ms = require("@sencinion/ms");
ms("1d");
```

## Özellikler

- Parse-ms kurulu
- Türkçe diline çevirilde
